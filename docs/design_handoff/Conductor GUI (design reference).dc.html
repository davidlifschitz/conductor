1 New GUI for Conductor — 4 directions

Conductor today is a read-only terminal dashboard (rich TUI): a status header, spend-by-model / spend-by-rule panels, and a live request tail — plus one-shot stats/tail/show commands. These four options bring that same data into a proper GUI, all keeping a dark, monospace-leaning "terminal, but polished" feel. Each is a real interactive shell — switch tabs, click a request row for detail, filter by model/rule, flip time ranges.
